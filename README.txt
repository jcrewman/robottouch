Robot touch study data

*.csv    -   continuous physiological sensor data, sampled at 32 Hz
marker*.csv   -   time markers to indicate the start of a trial, the participant'2s touch and the robot's action
stimuliOrder.csv   -   order of the trials
dat.Rdata   - summary data
alldat.Rdata   -   complete data structure
